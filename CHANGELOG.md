# Changelog

All notable changes to this package will be documented in this file.

## [1.0.0] - 2026-04-29

### Added

- Initial UPM package release.
- Push ScriptableObject field values to Notion databases.
- Pull Notion database values into ScriptableObject fields.
- `[TNotionDatabase]` and `[TNotionProperty]` attributes for declarative binding.
- `ITNotionSyncUniqueTable` interface for per-asset database IDs.
- Editor window (**Tools → Notion Syncer**) with search, multi-select, and console log.
- Custom property drawer showing a Notion icon on synced fields.
